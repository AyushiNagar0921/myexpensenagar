# Dream Tracker

## Overview

Dream Tracker is a personal finance management app that helps users:

- Track income and expenses
- Set savings goals
- Monitor loans
- Set up and manage budgets

## Tech Stack

- **React + Vite**
- **TypeScript**
- **Tailwind CSS**
- **shadcn/ui**
- **Supabase**
- **React Router**
- **Sonner (toast notifications)**
